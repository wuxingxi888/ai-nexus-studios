# 待测试

- 首页与控制台：首页恢复为静态极简首屏，移除未使用且会导致运行时报错的 ProConfigProvider，并补齐空图片封面兜底；需回归确认首页展示正常且控制台不再出现 ProConfigProvider 或空图片 src 相关报错。
- 首页 Hero 背景：hero 区默认使用封装后的 Lightfall 背景，刷新后随机显示 Lightfall、ShapeGrid 或 DotField 三种背景之一；需回归确认只影响首屏区域、按钮可点击且下方内容不受遮挡。
- 首页 Hero 内容：新增截图风格的居中胶囊状态、大标题、说明文案和双按钮内容层；需回归确认桌面端和移动端首屏比例、按钮点击和下方内容衔接正常。
- 首页案例区：首页调整为 Hero、图片案例、AI 漫剧三个板块，AI 漫剧板块接入 LingCut 作品鉴赏样例，支持 hover 内联播放、点击全屏预览和“查看制作过程”开发中提示；需回归确认图片预览、视频播放和移动端排版正常。
- 生图工作台 UI：页面按首页风格重构为顶部状态区、左侧创作控制舱、右侧结果画廊和历史记录模块，结果卡片、空状态和历史记录卡片同步调整为圆角玻璃视觉；需回归确认提示词、参考图、参数、生成、重试、下载、加入素材和历史记录行为不变。
- 视频任务创建协议：OpenAI 兼容视频创建统一走 JSON body，参考图按 `input_reference` URL 数组传递，任务查询读取 `/videos/{id}` 返回的 `task_result.videos[].url`；需回归确认 JSON-only 上游、参考图创建任务和视频结果下载正常。
- 视频模块收口：旧 `services/api/video.ts` 和 `lib/seedance-video.ts` 已移除，视频工作台、画布视频生成、Seedance 设置和素材编号工具统一走 `services/api/video/` 新模块；需回归确认生成、轮询、历史记录恢复、下载、加入素材和画布视频生成行为正常。
- 统一 AI 代理：渠道配置新增“浏览器直连 / Next.js 代理”请求方式，图片、文本、音频、视频和模型拉取会按渠道开关走直连或统一 `/api/ai-proxy`；Seedance 继续按非标准任务协议生成目标 URL。需回归确认跨域上游打开代理后可用。
- 渠道类型收口：渠道配置将“调用格式”和“视频协议”合并为“渠道类型”，支持 `OpenAI 兼容`、`Gemini 官方`、`Seedance 官方` 和 `自定义任务协议`；请求方式仍独立控制浏览器直连或 Next.js 代理。需回归确认模型拉取、图片、文本、音频和视频请求按渠道类型正常分流。
- 模型级渠道类型覆盖：渠道模型列表下新增模型渠道类型覆盖，可为单个视频模型覆盖渠道默认类型，分流优先级为“模型覆盖 > 渠道类型”，并移除自动识别协议；需回归确认 `cdance2.0-fast-0611` 可通过 UI 覆盖到 Seedance 官方，创建任务和轮询不再请求 `/v1/videos`。
- 自定义视频任务协议：选择自定义任务协议或将单个模型覆盖为自定义任务协议时显示自定义配置，可配置创建/查询路径、JSON Body 模板、任务 ID/状态/视频 URL/错误字段和状态映射。需回归确认自定义 JSON 任务型视频上游可创建、轮询并解析结果。
- 首页标题动效：标题渐变调整为白色主调和冷白闪光扫过效果；需回归确认深色背景上亮度、可读性和动画观感正常。
- 顶部导航悬浮玻璃态：未滚动时导航栏为全宽贴顶玻璃栏，滚动后下移到距离顶部约 50px 并收缩成悬浮玻璃态；需回归确认内容不被遮挡且移动端菜单正常。
- 顶部导航滚动源：导航栏依赖各页面主滚动容器上的 `data-nav-scroll-root` 标记，需回归确认首页、提示词、素材、生图、视频和画布列表页都能触发悬浮态。
- 顶部导航体验优化：导航结构拆分为品牌区、桌面导航和滚动状态 hook，滚动状态改为 DOM data 属性驱动，导航私有样式收口到组件 CSS Module，并通过 transform 弹性动效切换贴顶/悬浮状态；需回归确认动画顺滑、工作台顶部不被遮挡、滚回顶部不卡顿。
- 顶部导航玻璃态部署修复：导航玻璃背景改为独立伪元素承载 backdrop-filter，并移除可能影响生产合成层采样的 paint containment；需回归确认线上域名环境下导航不再只剩透明底。
- 全局滚动条：隐藏原生滚动条但保留滚动能力，需回归确认首页、工作台、弹窗、画布面板和横向素材列表仍可正常滚动。
- 首页 Lightfall 性能：Lightfall 离开视口附近后自动停止动画帧，恢复可见后再继续渲染，并限制 WebGL 像素比；需回归确认首页滚动性能和背景恢复正常。
- Docker 部署缓存：根布局改为动态响应并禁用页面 revalidate，避免宝塔/Nginx 反代在镜像升级后继续展示旧 HTML；需回归确认域名访问能看到新页面，且 `_next/static` 静态资源仍可正常加载。
- Web 前端结构：路由组从 `(user)` 调整为 `(workspace)`，页面业务迁移到 `web/src/features/`，需回归确认首页、画布、生图、视频、提示词库、我的素材和 WebDAV 代理路径行为不变。
- 配置与用户偏好：每个模型渠道新增渠道类型选择，默认 OpenAI 兼容，切换渠道类型时同步更新默认 Base URL；渠道 Tab 增加紧凑醒目的提醒，说明模型是否可选需要到“模型”Tab 选择。
- Gemini 官方渠道：支持拉取 Gemini 模型列表，并用于文本问答、在线 Agent 工具调用、基础生图和图生图请求。
- 音频和视频生成：选择 Gemini 官方渠道时先给出不支持提示，仍需使用 OpenAI 兼容、Seedance 官方或自定义任务协议渠道。
